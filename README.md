# denis-web-portfolio
WEB Portfolio de Denis Alonso Carrasco

## Starter Astro (portfolio de fotografia)
Starter limpio en Astro (sin backend) para un portfolio de fotografo/a, con paginas: Inicio, Portfolio, Sobre mi y Contacto. Contenido 100% en espanol.

### Requisitos
- Node.js 18+ (recomendado 20+)

### Comandos
```bash
npm install
npm run dev
npm run build
npm run preview
```

### Donde poner tus fotos
- Coloca imagenes en `public/images/`
- Actualiza los items en `src/data/portfolio.ts`
